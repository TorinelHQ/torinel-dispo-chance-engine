# Torinel Dispo Chance Engine

## 🇬🇧 English
This repository contains the scoring logic behind Torinel’s Dispo-Chance tool, used to estimate the likelihood of receiving an overdraft (Dispokredit) in Germany.

### Purpose
The model helps users make better financial decisions when choosing a Girokonto mit Dispo.

### Features
- Income-based evaluation
- Behavioral scoring signals
- Traffic light system (green / yellow / red)
- Recommendation and reranking logic

### Canonical implementation
👉 https://torinel.de/girokonto-mit-dispo/
